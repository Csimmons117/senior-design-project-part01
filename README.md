# CSUN CIT 480 Senior Project — Gym App Revamp (Web Version)
# [Website](https://csimmons117.github.io/senior-design-project-part01/)
## Overview
This project is our **senior design capstone** for CIT 480/CIT481 at CSUN.  
ToroForgeAI is a comprehensive website designed specifically for the Student Recreation Center (SRC) at California State University, Northridge. While the current SRC presence exists primarily as an informational page within the CSUN website, our platform transforms it into an interactive, user-centered experience. Students can view real-time busy hours, check operating hours, and securely register using their CSUN credentials. Once logged in, members can track their gym attendance through a monthly calendar, monitor progress with an experience-based tracking system, and visualize consistency over time. Additionally, our integrated AI trainer generates personalized workout plans and nutrition recommendations tailored to individual goals, including bulking, weight loss, or maintenance. This platform enhances engagement, motivation, and overall wellness for the CSUN community.

## System Development Lifecycle (SDLC)
We are following the SDLC as aligned with **CIT 480 course topics**:
1. **Requirements Gathering** → Stakeholder interviews, use cases, DFDs/BPMN  
2. **System & Data Modeling** → ERD, DB schema, process models  
3. **Design** → Architecture diagrams, UI prototypes, ADRs  
4. **Implementation (Phase I)** → Vertical slice (login, check-in, GPT coaching, admin basics)  
5. **Transition & Acceptance** → Deployment runbook, acceptance tests
